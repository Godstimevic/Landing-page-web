1. Discuss system software System software is a type of computer program that is designed to run a
computer's hardware and application programs. If we think of the computer system as
a layered model, the system software is the interface between the hardware and user
applications. Operating system like Windows, Android, Linux, and IOS are examples,
ensuring your computer functions smoothly. It's also the backbone that keeps everything in order.
